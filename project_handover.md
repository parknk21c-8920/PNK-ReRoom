## 1. 프로젝트 필수 웹사이트 (작업 시 열어두어야 할 4대장)

수정 및 배포, 관리를 위해 인터넷 창에 띄워두시면 좋은 사이트들입니다.

1. **내 실제 웹사이트 (Live):** https://pnk-re-room.vercel.app
   - 용도: 실제 고객의 입장에서 결과물과 시나리오를 테스트합니다.
2. **Supabase (데이터베이스):** https://supabase.com
   - 용도: 회원(고객) 관리, 가입자에게 크레딧(돈) 주기, 유저 정보 확인
   - 확인 위치: 해당 프로젝트 -> 왼쪽 메뉴 `Table Editor` -> `profiles` 테이블
3. **Vercel (배포 및 서버 관리):** https://vercel.com
   - 용도: "배포(Push)해줘"라고 했을 때, 성공적으로 인터넷에 업데이트 되었는지 확인합니다.
4. **Replicate (AI 엔진):** https://replicate.com
   - 용도: 인테리어 이미지를 그리는 AI 두뇌입니다. (사용된 비용 원가 확인)
5. **Stripe (결제 시스템):** https://dashboard.stripe.com/
   - 용도: 신용카드 결제 연동 및 실제 매출액을 확인하는 곳입니다.

## 2. 현재까지 완료된 작업 상태 (100% 달성)

* **UI/UX 구축:** 메인 페이지, 스타일 선택, 가격표(Pricing) 등 완벽한 반응형 웹(모바일+PC) 디자인 완성.
* **초고속 회원가입/로그인 (원클릭 프리패스):** 고객이 귀찮은 이메일 인증 절차 없이 1초 만에 가입 및 자동 로그인이 되도록 완벽한 편의성 구축 완료 (Supabase Service Role Key 적용).
* **상단 헤더 모바일 뷰 최적화:** 모바일 화면에서도 남은 크레딧(💳)이 보이도록 디테일 개선 완료.
* **AI 이미지 생성 (Replicate 연동):** 사용자가 사진을 올리면 AI가 공간을 분석해 인테리어를 생성하는 핵심 기능 구현 및 연동 완료.
* **Vercel 자동 배포:** 코드 수정 시 전 세계 어디서든 바로 접속할 수 있는 Live 서버에 즉각 배포되는 시스템 구축 완료.

## 3. 내일 이어서 할 작업 (Next Steps)

1. **최종 시나리오 실전 테스트:** 사장님 및 직원분들의 실제 이메일(네이버, 다음 등)로 `[가입 -> 로그인 -> 결제 페이지 -> 인테리어 생성]` 시나리오를 직접 타보며 사용자 경험(UX) 점검하기.
2. **결제 시스템 (PG) 실전 연동:** 현재 가짜 더미(Dummy)로 연결되어 있는 결제 버튼에, 실제 돈이 결제되는 Stripe 또는 국내 결제사를 연결하여 진짜 수익 창출 준비하기.
3. **상업화 도메인 연결 (선택):** `vercel.app` 이라는 임시 주소 대신, 돈을 주고 산 정식 도메인 주소(예: www.reroom.com)로 교체하여 신뢰도 높이기.
